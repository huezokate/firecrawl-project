## Summary

Build a web app called "What are devs building with Firecrawl?" that searches GitHub for public repos using Firecrawl as a dependency, scrapes their READMEs via Firecrawl, categorizes them via Claude, and displays results as a donut/bar chart + repo cards.

## Tech Stack

- **Backend**: Python / FastAPI, deployed to Railway
- **Frontend**: React, deployed to Cloudflare Pages
- **APIs**: GitHub Search API, Firecrawl API, Anthropic Claude API (claude-sonnet-4-20250514)

## User-Facing Features

- Donut/bar chart of category breakdown
- Repo cards: name, description, category tag, star count, link
- Clean minimal dark UI — developer tool aesthetic
