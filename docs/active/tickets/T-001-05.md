## Context

Build the React frontend. Fetches from the FastAPI backend, renders a donut/bar chart of category breakdown, and a grid of repo cards. Design: clean minimal dark UI, developer tool aesthetic — dark background, monospace accents, subtle borders. No CSS framework bloat — use Tailwind.

## Acceptance Criteria

- Vite + React + TypeScript + Tailwind CSS setup in `frontend/`
- `VITE_API_URL` env var controls backend URL
- Page layout: header → chart section → repo cards grid
- Chart: donut chart using Recharts showing category distribution; clicking a category filters the cards
- Repo cards: repo name (linked), description, language badge, star count, category tag (color-coded per category)
- Loading state while fetching; error state if API fails
- "Refresh data" button that calls `POST /api/refresh` then re-fetches
- Responsive: 1-col mobile, 2-col tablet, 3-col desktop grid
- Dark theme: `#0a0a0a` background, `#1a1a1a` cards, `#00ff88` accent, neutral grays for text
