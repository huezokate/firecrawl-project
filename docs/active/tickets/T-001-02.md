## Context

Implement the GitHub Search API integration. Search for public repos that list `firecrawl` as a dependency by querying `requirements.txt` and `package.json` file contents. Return structured repo data: name, description, stars, language, README URL, HTML URL.

## Acceptance Criteria

- `backend/github_client.py` with async function to search GitHub for firecrawl dependents
- Searches both `requirements.txt` (Python) and `package.json` (JS/TS) files
- Returns: repo full_name, description, stargazers_count, language, default_branch, html_url, readme_download_url
- Uses `GITHUB_TOKEN` from env for authenticated requests (higher rate limit)
- Handles pagination — fetches up to 100 results
- Unit-testable: no FastAPI coupling in this module
